# BỘ TÀI LIỆU BÀN GIAO KỸ THUẬT

## Danh mục tài liệu

1. [BAN_GIAO_DU_AN.md](./BAN_GIAO_DU_AN.md)
   Mô tả tổng quan dự án, phạm vi chức năng, công nghệ sử dụng và phạm vi bàn giao.

2. [KIEN_TRUC_HE_THONG.md](./KIEN_TRUC_HE_THONG.md)
   Mô tả kiến trúc hệ thống, thành phần frontend/backend, luồng dữ liệu và các module chính.

3. [TAI_LIEU_CSDL.md](./TAI_LIEU_CSDL.md)
   Mô tả cấu trúc cơ sở dữ liệu, các bảng chính, quan hệ dữ liệu và hướng dẫn khởi tạo DB.

4. [HUONG_DAN_DUNG_LOCAL.md](./HUONG_DAN_DUNG_LOCAL.md)
   Hướng dẫn dựng dự án trên máy local, cấu hình `.env`, Supabase, bàn giao Supabase và kiểm tra sau khi chạy.

5. [TAI_LIEU_KY_THUAT_TONG_HOP.md](./TAI_LIEU_KY_THUAT_TONG_HOP.md)
   Tài liệu kỹ thuật tổng hợp chi tiết — bao gồm toàn bộ API routes, database schema, state management, export features, deployment, biến môi trường, dịch vụ tích hợp và constants.

## Thứ tự gửi đối tác đề xuất

1. Gửi source code dự án.
2. Gửi toàn bộ thư mục `docs/`.
3. Gửi file `.env.example`.
4. Gửi thư mục `supabase/`.
5. Hẹn buổi hỗ trợ dựng local và rà lỗi môi trường.

## Gói bàn giao tối thiểu

- Mã nguồn hiện tại
- Tài liệu trong `docs/`
- File mẫu môi trường
- SQL schema
- Hướng dẫn dựng local
- Danh sách tài khoản/quyền truy cập cần chuyển giao riêng
